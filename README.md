# Valorant Database Browser
This browser was part of a project in Fresno State's Web Development Class.
In this browser, you can view some Valorant agents, along with their data. You may also add your own, or delete some. 
You may also organize the browser by category, or sort them alphabetically. 
The buttons on the bottom let you skip to the first or last agent.

# Example:
